# Mach3

Learn the operation interface for the Omio X6-2200 CNC.

![Mach3 Interface](/img/mach3-placeholder.jpg)

Mach3 is used to control the CNC. It allows you to "jog" and run G-code generated by the CAM software of your choice. Mach3 is not very beginner-friendly, as setting it up and configuring it to work with our CNC is very time consuming and finicky. However, it is a great program that gets the job done and works well most of the time if you know how to correctly use it and set it up.
